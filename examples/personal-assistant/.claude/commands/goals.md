# Goals

## Process
1. Load current goals and key results
2. If a specific goal is mentioned in $ARGUMENTS, focus on that one
3. Show progress for each key result — what's done, what's remaining
4. Ask if the user wants to update progress, add a new goal, or adjust targets
5. Save any changes

Goals span any domain — career, health, creative projects, relationships. Track by key results with progress notes, not percentages.
