# Session Check-In

## Process
1. Read the scratchpad (memory/) for last session's context — pending items, deadlines, current focus
2. Check todo lists for stale follow-ups (items with dates older than 7 days)
3. Review active goals for upcoming deadlines or milestones
4. Suggest 1-3 focus items using the priority framework: Deadlines > Quick Wins > High-Impact > Maintenance
5. Present a brief status and ask what the user wants to work on

Keep it concise. Don't dump the entire state — surface what matters today.
